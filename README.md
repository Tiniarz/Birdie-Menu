## 🚀 Features

Birdie Menu is packed with powerful features to help you dominate any game:

*   **Utility:** Essential tools to streamline your gameplay and optimize performance.
*   **Flight Settings:** Take to the skies with customizable flight speed and gravity toggles.
*   **Movement:** Advanced movement hacks, including speed boosts, infinite jumps, and air-strafe control.
*   **Visuals:** Gain the advantage with ESP, chams, and customizable tracers to track players easily.
*   **And More:** Constant updates and additional hidden features to discover!
